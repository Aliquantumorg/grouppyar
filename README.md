# 🤝 گروه‌یار — بات مدیریت گروه بله (ساختار صاف)

## 🛠 رفع باگ این نسخه (مهم)
قبلاً وقتی غیرادمین توی گروه «راهنما» می‌زد، فقط یه پیام در همون گروه می‌دید که بگه «برو پیوی استارت کن» — این دقیقاً چیزی نبود که خواسته بودی. الان اصلاح شد:

**رفتار درست:**
- **ادمین گروه یا مالک بات:** همون‌جا توی گروه، راهنما مستقیم نشون داده می‌شه (بدون تغییر)
- **عضو عادی (غیرادمین):** بات تلاش می‌کنه راهنما رو **مستقیم به پیوی خودش** بفرسته:
  - اگه قبلاً بات رو در پیوی استارت کرده باشه → راهنما (یا اگه عضویت اجباری فعاله، پیام عضویت) همون لحظه براش میره؛ توی گروه فقط می‌بینه «یه پیام توی پیوی برات فرستادم 📩»
  - اگه هنوز اصلاً پیوی بات رو باز نکرده باشه → بات (طبق محدودیت خود پلتفرم بله/تلگرام) اصلاً نمی‌تونه بهش پیام بده؛ در این حالت توی گروه بهش می‌گه «اول باید من رو در پیوی استارت کنی»

این محدودیت (نبود امکان پیام دادن به کسی که چت رو شروع نکرده) قانون خودِ پلتفرمه، نه محدودیت کد.

## ✅ تغییرات نسخه‌ی قبل (هنوز برقرارن)
- بعد از تأیید عضویت اجباری در پیوی، هم پیام «به گروهت اضافه کن»، هم کل راهنما ارسال می‌شه
- نام لحن (مثل «لحن: چاله‌میدونی») از بالای راهنما حذف شد
- منطق راهنما توی `helptext.py` جداست (برای جلوگیری از ایمپورت چرخه‌ای بین `main.py` و `force_join.py`)
- سیستم گیف و سیستم سکه کاملاً حذف شدن
- دستور «لغو بازی» اضافه شده

## ⚠️ نکته‌ی امنیتی (قبل از push به گیت‌هاب)
توکن واقعی بات رو هرگز داخل `config.py` ننویس و commit نکن؛ از متغیر محیطی `BOT_TOKEN` استفاده کن. `.gitignore` پیشنهادی:
```
__pycache__/
*.pyc
gapbot.db
gapbot.db-*
.env
```

## 🧪 تست دقیق این نسخه
1. با یه اکانت که **قبلاً هیچ‌وقت پیوی بات رو باز نکرده**، توی گروه «راهنما» رو بزن → باید بگه «اول پیوی استارت کن».
2. همون اکانت رو ببر پیوی بات، «شروع» رو بزن (و اگه عضویت اجباری فعاله، رد کن).
3. حالا با همون اکانت، دوباره توی گروه «راهنما» رو بزن → این‌بار باید بگه «توی پیوی برات فرستادم» و واقعاً توی پیوی راهنما رو ببینی.
4. با اکانت ادمین، توی گروه «راهنما» رو بزن → باید مستقیم همون‌جا راهنما رو ببینی.

هر ارور دیدی، کل متن ترمینال رو بفرست.
